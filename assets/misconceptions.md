# 한국인 학습자 오개념 카탈로그 (코퍼스 trunk 하위 자산)

R2(근접 오개념 오답)·R3(경계 문항)의 **설계 재료 정본**. 매 사이클 검수·페르소나·실사용에서 발견된 오개념을 여기 회수 — 새 콘텐츠는 이 카탈로그를 입력으로 오답·경계 문항을 설계한다 (content-consensus 검수 관점 3이 대조).

형식: 오개념 → 교정 감각 → 증거(문항·검수 ID).

## 번역 등치 (감각 소실의 뿌리)

- **have = 가지다** → 영역 안 정적 위치 (소유·구매 아님) — have-t1~t5
- **get = 얻다/노력해서 획득** → 밖에서 안으로 들어와 닿는 도달 (수령 포함, 노력 무관) — get-t2 오답 설계
- **take = 취하다** → 손 뻗어 자기 영역으로 끌어오는 의지적 점유 — take vs get 대비 문항
- **keep = 유지하다** → 흘러나가려는 것을 힘으로 붙듦 (have와 force 유무로 갈림) — keep-t5·t6
- **up = 위** → 수직 + 완료·한계 도달 (eat up, time's up) — up-t6~t10, 페르소나 "평생 '위'로만 알던"
- **out = 밖** → 안→밖 + 드러남(find out) — out-reveal 계열

## 문법 행동 오개념

- **정적 have의 진행형** (*I am having a sister) ↔ 사건성 have는 가능 (are having lunch) — boundary_ko 표지, c1-1 검수 B2
- **be vs get 상태** (was dark = 상태 / got dark = 변화 진입) — get-x4 (검수 M2로 문장 교정)
- ***make homework*** — do(행위) vs make(새로 생겨남) 한국인 최대 혼동 — make-t7
- **keep vs stay** — stay는 스스로 머묾, keep은 붙드는 힘 — keep-t8

## 구동사 양극단 (둘 다 오개념)

- **"구동사 = 통암기 숙어"** (합성 불인지) → 다수는 두 그림의 합 (get up, drink up) — phrasal-up compose 계열
- **"모든 구동사 = 그림의 합"** (합성 과신) → give up·look up(찾다)·make up(지어내기)은 관용 — R10, 페르소나 "give up에서 깨질 것", opaque-idiom sense

## 방향·은유 오개념

- **bring/take 동일시** → deixis: bring은 화자 쪽으로, take는 화자에게서 멀어짐 — take-carry (검수 M4)
- **상태 변화의 방향** → 주어가 상태(위치)로 옮겨 감 (상태가 다가오는 게 아님 — STATES ARE LOCATIONS) — get-state-change (검수 B3)

## 운영 규칙

- 신규 오개념은 발견 사이클에서 즉시 이 파일에 1줄 회수 (cycle record에만 남기면 미회수).
- 콘텐츠 생성 프롬프트는 이 파일을 의무 입력으로.
- 거부 기록: "감각-한국어 대조 레퍼런스" 별도 자산은 거부 (retro-2) — 이 카탈로그의 번역 등치 섹션이 역할 흡수, 중복 자산은 비대화.
