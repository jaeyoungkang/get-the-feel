# Process And Monitor Recovery

Required monitors for this product:

- Intent Guardian: original macro value-chain ask.
- Asset Steward: reusable assets and skills are growing.
- Data/Sellability Monitor: source contracts, freshness, future scenarios, sellability truthfulness.

Monitor feedback must become one of:

- skill update,
- asset update,
- mechanical check,
- product code/data contract,
- cycle record gate.
