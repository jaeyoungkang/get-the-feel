# Visualization And UX

## Visual Grammar

- Left to right chain: inputs -> silicon -> integration -> infrastructure -> platform -> demand/revenue feedback.
- Nodes are value-chain stages.
- Edges are pressure transmission.
- Size/color/edge thickness are bottleneck pressure.
- Snapshot controls update the same map.
- Detail panel is macro-to-micro drilldown, not primary content.

## Reject Patterns

- Four cards above the map.
- Company cards as first viewport.
- Table-first research terminal.
- Labels that require reading notes before the map makes sense.
