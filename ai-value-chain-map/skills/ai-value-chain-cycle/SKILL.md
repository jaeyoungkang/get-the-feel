# Goal

Build the AI value-chain map while growing reusable assets.

## When To Use

- Before any product cycle in `ai-value-chain-map`.
- Before changing the map, data contracts, business logic, or cycle gates.
- Before representative or sellable promotion.

## Workflow

0. **Skill Load Receipt** — record this skill path, loaded step, required gates, and blockers.
1. **Asset Preflight** — read `assets/ASSET_MAP.md`; name the asset updated this cycle.
2. **Original Ask Anchor** — first viewport must remain macro AI value-chain map for ordinary investors.
3. **Data Contract Gate** — no displayed value without source/status/time/freshness/snapshot/calculation contract.
4. **Scenario Boundary** — future snapshots must show assumptions and must not be labeled observed.
5. **Sellability Gate** — paid release stays blocked until source workflow, owner, cost, formula, and advice boundary are known.
6. **Mechanical Gate** — run `npm run quality:check`.

## Never

- No stock recommendation surface.
- No company-card primary UI.
- No `observed` label for curated indexes or scenarios.
- No monitor feedback left only in chat.
