# Visualization And UX

## Visual Grammar

- Map first, cards second.
- Nodes represent supply-chain stages.
- Edges represent pressure propagation.
- Node size and color represent bottleneck pressure.
- Edge thickness and color represent transfer pressure.
- Snapshot controls change the same map, not a separate report.

## Macro-To-Micro

Clicking a stage opens micro components, watch points, data status, and contract visibility.

## Reject Patterns

- Four summary cards as the main product.
- Company dossier as the main product.
- Text explanation that claims the map works while the map itself is shallow.
- Visual changes that do not change the user's reading path.
