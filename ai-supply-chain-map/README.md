# AI Supply Chain Map

일반 주식투자자가 AI 공급망의 거시 병목을 보고, 선택한 구간의 미시 정보와 시간 변화/미래 시나리오를 확인하는 정적 웹 프로토타입.

## 실행

```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173/ai-supply-chain-map/`를 연다.

## 현재 범위

- SVG macro supply-chain map
- 병목 압력 edge와 node 표시
- macro-to-micro detail panel
- 과거/현재/미래 시나리오 스냅샷
- 공식 출처 링크

투자 조언, 종목 추천, 목표가 산출 도구가 아니다.
