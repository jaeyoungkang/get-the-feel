# Product-Weaver Era Archive

이 폴더는 get-the-feel을 프로토타입으로 탐색하던 시기의 운영 문서를 보존한다.
현재 제품 개발 기준은 `docs/contracts/story-chain/`와 Mission Control이다.

| File | Historical role |
| --- | --- |
| `principles.md` | historical methodology 시절의 원리 문서 |
| `asset-map.md` | 프로토타입 자산 지도 |
| `project-elevation-plan.md` | 프로토타입에서 정식 프로젝트로 승급하던 전환 계획 |
| `domain-layer-activation-guide.md` | historical methodology 도메인 레이어 활성화 가이드 |

새 작업은 이 문서를 직접 기준으로 삼지 않는다. 필요한 판단은 Story Chain
Promise, Aspect, Evidence Ledger, product contract, content contract로 회수한다.
