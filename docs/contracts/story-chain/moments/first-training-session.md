# First Training Session

사용자는 제품을 열고 별도 설명 없이 한 세션을 시작한다. 문항을 풀면서 정답 전에는 단서가 노출되지 않고, 정답 후에는 문장 해석·감각 그림·간단한 해설을 통해 자신이 어떤 영어 감각을 맞혔거나 놓쳤는지 확인한다.

세션이 끝나면 사용자는 새 문장 테스트와 통계 표면을 통해 어떤 감각이 아직 약한지 보고, 바로 같은 감각을 집중해서 연습할 수 있어야 한다.
